## Features

### 4 Role-Based Dashboards

- **Technician** — Progress ring, section cards with expandable task tables, portfolio checklist, pipeline status
- **Evaluator** — Candidate roster, task sign-off workflow with digital signatures
- **Committee** — Oral board queue, 3-phase rubric scoring (Technical Knowledge, Practical Application, Professional Standards)
- **Admin** — Pipeline funnel, all-candidates table, evaluator quality metrics, program statistics

### Certification Pipeline

1. **Written Exam** → Knowledge assessment
2. **PQS Tasks** → 75 hands-on practical tasks across 13 sections
3. **Portfolio** → 7 evidence items documenting real-world work
4. **Oral Board** → 3-phase committee review
5. **Certified** → CAET Advanced credential awarded

### Key Capabilities

- Real-time progress tracking across all 75 tasks
- Section-by-section completion with visual progress bars
- Evaluator sign-off workflow with timestamps
- Portfolio evidence checklist
- Oral board rubric scoring (1-5 scale)
- Pipeline funnel visualization
- Evaluator quality metrics
- Zero external dependencies — single-file HTML
- Fully functional without JavaScript (JS enhances only)
